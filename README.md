### 增量存储，不会撑爆硬盘

很多人担心：存档多了是不是很占地方？

追光 Lite 采用增量存储技术，每次存档仅保存文件变化差异的那一小部分，而非完整复制一份。比如一个 100MB 的文件，第一次修改了 10KB，第二次改了 5KB，三次存档总共也只占 100MB+15KB 左右。相比手动复制三份 300MB，省了一大截，文件夹也不会乱成一锅粥。

特别适合那种"我记得之前有一个方案方向，后来废掉了，现在想找回来"的情况。
